house-monitor

This is the software for a mechanism I built to monitor my house while I'm away.

It runs on a raspberry pi 1 and uses a one wire temperature sensor and raspberry pi camera .

It uploads the data to a cloud service so I can access it from anywhere.

The client application is an Angular2 app written in typescript.
The server and misc utilities are all written in Node.js. The temperature graph is done with google charts and nginx handles http.
